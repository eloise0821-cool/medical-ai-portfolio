
## GitHub Pages 上线

1. 在 GitHub 新建一个仓库，例如 `medical-ai-portfolio`。
2. 将本项目文件推送到仓库默认分支。
3. 打开仓库 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 GitHub Pages 生成公开链接。

## 上线前建议替换

- `index.html` 中的 GitHub 链接
- 如有 PDF 简历，可添加到仓库并把“下载简历”按钮指向该文件
- 如希望公开网页更谨慎，可将手机号替换为邮箱或微信
